# 🚀 도메인 로직

1. 금액 입력 받기
2. 구매 개수만큼 중복 없는 `6`개 숫자 뽑기
3. 랜덤으로 뽑힌 각 로또 번호 출력
4. 당첨 번호, 보너스 번호 입력받기
5. 당첨 결과, 수익률 출력
