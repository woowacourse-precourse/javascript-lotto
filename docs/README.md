# 로또

##  🚀 만들어야하는 기능

- [ ] 로또 금액 이력 받기
  - [ ] 1000원 단위로 나누어떨어지지 않는 경우 예외처리
- [ ] 당첨 번호 입력 받기
  - [ ] 쉼표 기준으로 나눴을 때 6개 숫자인지 확인
  - [ ] 각 숫자의 범위가 `1~45`인지 확인
- [ ] 보너스 번호 입력받기
  - [ ] 숫자의 범위가 `1~45`인지 확인하기
- [ ] 입력 받은 숫자들이 중복되지 않았는지 확인하기.
- [ ] 로또 금액으로 로또 발행하기
  - [ ] 랜덤으로 겹치지 않는 숫자 7개 뽑기
- [ ] 당첨 내역 출력하기
  - [ ] 각 로또 번호와 당첨 번호를 비교하여 몇 개의 숫자가 맞았는지 체크
- [ ] 총 수익률 계산하기
