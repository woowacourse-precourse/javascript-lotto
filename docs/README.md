# 구현 시나리오

- [✔] 1. 로또 구입 금액 입력받기
- [✔] 2. 발행한 로또 수량 및 번호 출력(오름차순 정렬, 로또 1장 = 1000원)
- [✔] 3. 당첨 내역 및 보너스 번호 출력(몇개의 일치 숫자 및 당첨 금액 출력)
- [✔] 4. 수익률 출력(소수점 둘째 자리에서 반올림)
- [✔] 5. 게임 종료

# 구현 특이사항

- [] 로또 구입 금액 Validation 처리
  - [] 아무 입력 값이 없을 경우
  - [] 1000원으로 나눠지지 않는 경우
  - [] 0 혹은 음수의 숫자
  - [] 숫자가 아닌 경우
