# 구현 로직

1. 구매 금액 입력받기
2. 입력받은 금액이 1000으로 나누어 떨어지지 않으면 에러 출력
3. 1~45 중 중복 없는 6개 숫자 뽑기 + 오름차순 정렬
4. 3번 구매 개수만큼 반복해서 출력
5. 당첨 번호 6개 입력 받기
6. 보너스 번호 하나 입력 받기
7. 3/4/5/6 일치하는 개수 찾기
8. 당첨 금액/투자 금액으로 수익률 계산
9. 당첨 내역과 수익률 출력 후 게임 종료
   <br/><br/>

# 구현 기능

- [x] 구매 금액 입력받기
- [x] 구매 금액 에러 처리
- [x] 랜덤 숫자 6개 뽑기
- [x] 오름차순 정렬
- [x] 발행한 로또 출력
- [x] 당첨 번호 입력받기
- [x] 보너스 번호 입력받기
- [x] 당첨 번호와 로또 번호 몇 개 일치하는지 확인
- [ ] 수익률 계산
- [ ] 결과 출력
- [ ] 게임 종료
