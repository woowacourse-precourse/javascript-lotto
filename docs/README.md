## 미션 - 로또

#### 미션 목표
- 기능 요구사항, 프로그래밍 요구 사항, 과제 진행 요구 사항 세 가지를 모두 충족하며 진행한다.
- 2주차 과제 시 느꼈던 문제점을 개선한 코드를 작성한다.
  - `중복 코드`가 발생하지 않도록 코드 작성
  - `변수명` 구체화
  - 필요 시 `클래스` 활용
- `Jest`를 이용하여 테스트 코드 활용
- if 조건절에 return을 활용하여 `else의 사용을 지양`하자
  단, if/else, switch문의 활용이 직관적일 경우 사용 가능
