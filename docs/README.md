## 기능 구현 목록

1. App Class

- [x] 로또 구입 금액 입력 받기
  - [x] 1000원으로 나누어 떨어지지 않는 경우, 에러 던지기
- [x] 구매한 로또 번호 출력
- [x] 당첨 번호 입력 받기
- [x] 보너스 번호 입력 받기
- [x] 로또 당첨 통계 출력하기
  - [x] 수익률은 소수점 둘째 자리에서 반올림

2. Lotto

- [x] 로또 번호 출력하기
- 발급 받은 로또 검증
  - [x] 중복 유무
  - [x] 숫자의 범위
  - [x] 번호 갯수

3. Lotto Machine Class

- [x] 입력 받은 구입 금액만큼의 로또 발급하기
  - [x] 로또 번호는 랜덤으로 추출

4. 에러 정리

- [x] 모든 에러 문구는 `[ERROR]`로 시작하고 어떤 내용이 문제인지 출력
