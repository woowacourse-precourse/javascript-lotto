## 기능 구현 목록

> ### [/] 로또 구입 금액을 입력 할 수 있다

- [/] 숫자는 1000으로 나누어 떨어져야 한다.
- [/] 입력한 금액에 대한 메세지 를 출력한다.

> ### [/] 입력된 숫자 만큼의 로또 번호 랜덤 생성

- [/] 로또의 숫자 범위는 1~45까지이다.
- [/] 번호는 오름차순이다.
- [/] 생성된 로또를 출력한다.

> ### [/] 당첨번호를 입력 할 수 있다.

- [/] 보너스 번호를 입력 할 수 있다.
- [/] 당첨번호와 보너스 번호를 출력한다.

> ### [/] 당첨결과를 저장할 클래스를 만든다.

- [/] 정답 평가가 이루어 질 때마다 클래스의 필드값을 업데이트한다.

> ### [ ] 구입한 로또와 당첨번호를 비교하여 결과를 출력 한다.

- [ ] 맞춘 갯수에 대한 당첨 통계 (1~5등) 를 출력 한다.
- [ ] 투입한 금액 대비 수익률을 계산하여 소수점 둘재 자리에서 반올림한다.
