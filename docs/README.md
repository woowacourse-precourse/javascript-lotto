# 기능 목록

## App

| 분류 | 기능 | 예외사항 |
|:--:|------|-------|
| UI | 구입 금액 입력 및 검사 | 입력값이 자연수가 아닐 경우 <br/> 입력값이 1000의 배수가 아닐 경우 |
| 핵심 로직 | 금액에 맞게 로또 구매 ||
| UI | 구매한 로또 번호 출력 ||
| UI | 당첨 번호 입력 | 입력값의 구분자가 `,` 가 아닐 경우 |
| UI | 보너스 번호 입력 ||
| 핵심 로직 | 각 로또별 당첨금 계산 ||
| 핵심 로직 | 당첨 통계 계산 ||
| UI | 당첨 통계 출력 ||

## Lotto

| 분류 | 대기능 | 소기능 | 예외사항 |
|:---:|----|------|---------|
| 핵심 로직 | 로또 발행 | 번호 자동 선택 ||
