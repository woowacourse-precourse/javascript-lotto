## 🗒️ **기능 구현 List**

- 금액만큼의 로또 번호 발행 기능(1~45 숫자 중 중복되지 않는 6개의 숫자)
- 발행한 로또 번호 정렬 기능
- 당첨 번호 입력 기능(1~45 숫자 중 중복되지 않는 6개의 숫자)

- 수익률 구하는 기능(소수점 2째 자리에서 반올림)
- 입력값이 필요한 기능들에 대한 예외 처리
- 발행된 로또 번호 출력 기능
- 당첨 내역 출력 기능
