# 구현할 기능 목록 및 흐름

- [x] 사용자 로또 구매 금액 입력
- [x] 1 ~ 45 사이의 로또 당첨 번호 생성
- [x] 사용자 로또 번호 입력
- [x] 사용자 보너스 번호 입력
- [x] 사용자가 입력한 번호와 로또 당첨 번호 비교 및 출력

# 예외 처리

- [x] 구입 금액 입력 시 1,000 단위가 아닌 경우
- [x] 구입 금액 입력 시 숫자가 아닌 경우
- [x] 구입 금액이 1,000 이하인 경우 (1개도 구매하지 않은 경우)

- [x] 로또 번호 중복 제거
- [x] 로또 번호 갯수 확인

- [ ] 당첨 번호 입력은 쉼표(,) 기준 구분
