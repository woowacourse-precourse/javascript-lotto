## 입력값 (당첨 번호)

- 정수이어야 한다.
- 1 ~ 45 사이의 정수여야 한다.
- 중복되지 않아야 한다.
- 7개를 뽑아야 한다. (일반 숫자 6 + 보너스 숫자 1)
- 쉼표를 기준으로 구분한다.

## 입력값 (로또 구입)

- 로또 구입 금액을 받는다.
- 1000원으로 나누어 져야 한다.

## 출력값

- 로또 구매 개수 출력
- 오름차순으로 정렬
- 당첨 내역 출력
- 수익률 출력 (소수점 둘째 자리에서 반올림)
- 에러 상황 "[ERROR] 로또 번호는 1부터 45 사이의 숫자여야 합니다."

## 실행 순서

1. 구입 금액 입력
2. 구매 장 수 출력
3. 구매한 로또 번호 랜덤 돌려서 출력
4. 당첨 번호 입력
5. 보너스 번호 입력
6. Lotto 클래스에 전달
7. 계산
