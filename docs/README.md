# 로또게임 제작
--------
## 큰틀?
    크게 네가지의 기능구현을 하면 될것이라 생각합니다.
    1. 금액을 입력하면 로또를 주는기능
    2. 숫자 6개와 1개를 배열로 만든다.
    3. 로또를 하나씩 확인하며 등수를 확인하는 기능
    4. 등수를 보여주고 수익률을 보여주는 기능
    5. 각자 예외는 throw로 보내기

## 1번 기능
```
1. 금액을 입력하는 기능
2. 입력한 금액을 확인하여 제대로된 입력인지 확인
3. 1000원의 배수만큼 로또번호 출력
```
## 2번 기능
```
1. 로또번호와 보너스 번호를 입력
2. 제대로된 번호인지 확인
```
## 3번 기능
```
1. 로또 번호와 보너스 번호를 기준으로 반복문으로 등수확인
2. 수익률 확인
```
