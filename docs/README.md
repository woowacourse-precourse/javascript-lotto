### 구현 기능

1. 로또 구입 금액 입력
2. 금액 Validation Test
   - 1000 단위로 나누어 떨어지지 않는 경우
   - 금액이 숫자가 아닐 경우
3. 로또 번호 출력
4. 당첨 번호 6자리 입력
5. 당첨 번호 Validation Test
   - 1~45 범위의 숫자가 아닐 경우
   - 중복된 숫자일 경우
   - 당첨 번호가 숫자가 아닐 경우
6. 보너스 번호 입력
7. 보너스 번호 Validation Test
   - 1~45 범위의 숫자가 아닐 경우
   - 중복된 숫자일 경우
   - 보너스 번호가 숫자가 아닐 경우
8. 당첨 결과 출력

### 로또 기능

1. 랜덤한 숫자 6자리 생성
2. 당첨된 등수 산출
3. 금액 계산
4. 통계 출력
