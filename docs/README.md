2. 복권 갯수와 금액에 맞는 복권 랜덤 배열 출력 기능

3. 플레이어 당첨 번호 입력 

4. 보너스 번호 입력 및 

5. 당첨 번호와 복권 비교 기능

6. 비교값에 대한 통계자료 출력 및 어플리케이션 종료
