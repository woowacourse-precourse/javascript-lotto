# 기능 목록

### 주요 기능

- [x] 로또 구입 금액 입력받기
- [x] 당첨 번호 입력받기
- [x] 보너스 번호 입력받기

- [x] 발행한 로또 수량, 번호 출력
- [x] 당첨 내역 출력하기
- [x] 수익률 출력하기

### 예외 처리

- [x] 로또번호 입력 예외
  - 1-45 사이가 아닌 경우
  - 숫자가 아닌 값을 입력한 경우
  - 쉼표(,) 로 구분하여 입력하지 않은 경우
  - 서로 다른 숫자가 아닌 경우
- [x] 보너스번호 입력 예외
  - 1-45 사이가 아닌 경우
  - 숫자가 아닌 값을 입력한 경우
- [x] 구입 금액 입력 예외
  - 숫자가 아닌 경우
  - 1000원 단위가 아닌 경우
